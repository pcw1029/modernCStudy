# modernCStudy
모던C 프로그래밍 학습

# 1. 환경정보

* ``Ubuntu 18.04.x``
* ``googletest v3.10``
* ``Cmake 3.10.2``


# 2. 책 정보
* ``책에 나온 예제 다운로드 페이지 : https://wikibook.co.kr/modern-c-programming/``
* ``이 책에서 C99의 일부 사양을 이용하고 있다. 따라서 이클르스에 C/C++ Build 설정에 std=c99 옵션을 추가해야 한다.``


# 3. Cmake 설치
<pre>
mkdir build
cd build
sudo cmake ..
sudo make
sudo make install 
</pre>
